# Majoo Golang Bootcamp

## Setup project
 - Clone project
 - Run command
    ```
    go mod tidy
    ```

## Run project

 - Go to root folder
 - Run command
    ```
    go run ./cmd/main.go
    ```
- Server will run in `localhost:8000`

## API List
 - User
    - POST v1/login
    - POST v1/register
    - POST v1/user
    - PUT v1/user
    - DELETE v1/user/:ID
 - Outlet
 - Category
 - Product
 - Transaction
    - GET v1/transaction
    - GET v1/transaction/:ID
    - POST v1/transaction
    - PUT v1/transaction
    - POST v1/transaction/payment
